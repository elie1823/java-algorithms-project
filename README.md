#  Projet Java : Algorithmes de Tri et de Recherche

##  Description
Ce projet est une implémentation en Java de plusieurs algorithmes de base, notamment les algorithmes de tri et de recherche.  
Il a été réalisé dans le cadre d’un travail scolaire afin de comprendre la logique des algorithmes et la structure du langage Java.

---

##  Fonctionnalités

###  Algorithmes de tri
- Tri à bulles
- Tri par sélection
- Tri par insertion

###  Algorithmes de recherche
- Recherche linéaire
- Recherche binaire

---

##  Fonctionnement

Le programme permet de :
- Trier des tableaux de nombres
- Rechercher un élément dans un tableau
- Afficher les résultats dans la console

---

## Structure du projet
src/
├── Main.java
├── Sorting.java
└── Searching.java

## ▶ Comment exécuter le projet

1. Ouvrir le projet dans un IDE (IntelliJ, Eclipse ou VS Code)
2. Compiler les fichiers Java
3. Exécuter le fichier `Main.java`

##  Exemple de sortie

Avant tri :
`[5, 2, 9, 1, 3]`

Après tri à bulles :
`[1, 2, 3, 5, 9]`

Résultat recherche :
- Index de l’élément trouvé ou -1 s’il n’existe pas
